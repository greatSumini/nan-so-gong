# Nan so gong

간단한 블록체인 지갑입니다.

2022년 1학기 소프트웨어공학 수업의 과제 제출물입니다.

## 🙋 시작해볼까요?

### 1. 저장소를 기기에 Clone합니다.

```shell
$ git clone https://github.com/greatSumini/nan-so-gong.git

$ cd nan-so-gong
```

### 2. 의존성 설치

Yarn을 사용합니다!

```shell
$ npm i -g yarn

$ yarn install
```

### 3. 실행 🏃

```shell
# 개발 서버 실행
$ yarn dev

# 검사 (lint, 타입체크를 포함합니다.)
$ yarn run check
```
